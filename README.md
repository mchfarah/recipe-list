# Recipe List | Lista de Receitas

## 🇺🇸 English

A web page with culinary recipe cards.
Built to practice semantic HTML and modern CSS.

### Technologies

- HTML5
- CSS3 (Grid, Flexbox)

### How to run

Open the `index.html` file in your browser.

### What I learned

- Semantic structure with header, main, article, footer
- CSS Grid for responsive layout
- Card styling with hover effect
- File organization in folders
