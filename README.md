# Recipe List | Lista de Receitas

## English

A web page with culinary recipe cards, each with its own detail page including ingredients and step-by-step instructions. Built to practice HTML, modern CSS and JavaScript.

### Live Demo

[mchfarah.github.io/recipe-list](https://mchfarah.github.io/recipe-list)

### Technologies

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- JavaScript (DOM manipulation, event listeners)

### How to run

Open the `index.html` file in your browser or access the live demo above.

### What I learned

- Semantic HTML structure with header, main, article, footer
- CSS Grid for responsive layout
- Card styling with hover effects
- CSS Variables for consistent theming
- Multi-page navigation with relative paths
- Category filter with JavaScript
- Deploying a project with GitHub Pages

## Português

Página web com cards de receitas culinárias, cada uma com sua própria página de detalhes contendo ingredientes e modo de preparo. Desenvolvido para praticar HTML semântico, CSS moderno e JavaScript.

### Demo

[mchfarah.github.io/recipe-list](https://mchfarah.github.io/recipe-list)

### Tecnologias

- HTML5
- CSS3 (Grid, Flexbox, Variáveis CSS)
- JavaScript (manipulação do DOM, event listeners)

### Como visualizar

Abra o arquivo `index.html` no navegador ou acesse a demo acima.

### O que aprendi

- Estrutura semântica com header, main, article, footer
- CSS Grid para layout responsivo
- Estilização de cards com efeito hover
- Variáveis CSS para consistência de cores
- Navegação entre múltiplas páginas com caminhos relativos
- Filtro por categoria com JavaScript
- Deploy de projeto com GitHub Pages
