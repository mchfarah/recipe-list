About this project

This is my first project where I decided to truly test my skills and reach a satisfying result, something useful that I can always add more content to, put into practice what I've been studying, and let creativity flow. Built with HTML, CSS and JavaScript, it features recipe cards with detail pages and category filtering, for now.

Esse é de fato o primeiro projeto onde decidi testar minhas habilidades e chegar a um resultado satisfatório, algo útil onde posso sempre adicionar mais conteúdo, colocar em prática o que venho estudando e deixar a criatividade fluir. Desenvolvido com HTML, CSS e JavaScript, conta com cards de receitas com páginas de detalhes e filtro por categoria, por enquanto.
